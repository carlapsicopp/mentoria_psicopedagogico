# Landing Page - Mentoria

Landing page simples com integração de pagamento via Mercado Pago, hospedada no GitHub Pages.

## Sobre o projeto

Página de vendas de mentoria com valores fixos e redirecionamento direto para checkout do Mercado Pago. Não processa dados de cartão — todo o pagamento é feito na página oficial do Mercado Pago.

## Estrutura

```
├── index.html          # Página principal
├── css/
│   └── style.css        # Estilos
├── js/
│   └── script.js         # Evento do botão de compra
├── img/                  # Imagens e ícones
└── README.md
```

## Tecnologias

- HTML5
- CSS3
- JavaScript (vanilla)
- Hospedagem: GitHub Pages
- Pagamento: [Mercado Pago](https://www.mercadopago.com.br/) (Link de Pagamento)

## Pagamento

O botão de compra redireciona para um link de pagamento gerado no Mercado Pago, com valor fixo definido no momento da criação do link. O cliente escolhe a forma de pagamento (Pix, cartão ou boleto) diretamente na página do Mercado Pago.
